README_src.txt for version 9.2 of MNV: MNV is not Vim.

The source archive contains the files needed to compile MNV on Unix systems.
It is packed for Unix systems (NL line separator).

For more information, see the README.txt file that comes with the runtime
archive (mnv-9.2-rt.tar.gz).  To be able to run MNV you MUST get the runtime
archive too!
